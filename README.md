```shell
cd build
cmake ..
make
```

```shell
./batch_test.sh
```

## dev

`git branch`可以查看当前分支。用`checkout`命令切换到自己的分支（姓名首字母缩写）进行开发。

使用`git push origin <dev>`推送到自己的开发分支。

可以用`git merge <dev>`将开发分支合并到master分支。

更多参考[这篇博客](https://blog.csdn.net/bufegar0/article/details/109597640)
